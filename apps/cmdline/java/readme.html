Native teegi kompileerimine

Windowsil avage apps/cmdline/project/vc2010/cmdline.sln ja kompileerige java projekt
Unixil käivitage kataloogis apps/cmdline/project/unix make -f Makefile.java cpp

Java liidese ja testprogrammi kompileerimine

Windowsil kompileerige apps/cmdline/java kataloogis olevad java failid
Unixil käivitage kataloogis apps/cmdline/project/unix make -f Makefile.java java

Kasutamine

  • Kopeerige kompileeritud native teek (morph_jni.dll või libmorph_jni.so) süsteemi teele.
  • Kopeerige et.dct kompileeritud Java klasside juurde.
  • Käivitage MorphTest andes parameetriteks analüüsitavad sõnad. NB! mõni konsool ei pruugi hakkama saada täpitähtedega.
  • Muutke soovi korral MorphTest.java faili seadistamaks tarkvara tööd.
