Native teegi kompileerimine

Windowsil avage apps/cmdline/project/vc2010/cmdline.sln ja kompileerige *_jni projektid
Unixil käivitage kataloogis apps/cmdline/project/unix make -f Makefile.java cpp
Vajadusel kohendage projektifailides kasutatav JDK kataloog oma Java installatsioonile vastavaks.

Java liidese ja testprogrammi kompileerimine

Windowsil kompileerige apps/cmdline/java kataloogis olevad java failid
Unixil käivitage kataloogis apps/cmdline/project/unix make -f Makefile.java java

Kasutamine

  • Kopeerige kompileeritud native teegid (*.dll või *.so) süsteemi teele.
  • Kopeerige et.dct ja et3.dct kompileeritud Java klasside juurde.
  • Käivitage LinguisticTest andes parameetriteks analüüsitavad sõnad või käivitage DisambiguatorTest andes parameetriks tühikutega eraldatud lause. NB! mõni konsool ei pruugi hakkama saada täpitähtedega.
  • Muutke soovi korral *Test.java faile seadistamaks tarkvara tööd.
